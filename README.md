# Hardware test

Software for hardware testing.

## Requriements
- linux
  - upower (for batteries detect)
  - lshw (for graphics detect)

## Commands:
- ### install packages
        npm install

- ### start app
    - #### for normal user:
            npm start
    - #### for root user:
            npm run startRoot

- ### build app
    - #### linux
            npm run build-lin
    - #### windows (not working propertly)
            npm run build-lin
